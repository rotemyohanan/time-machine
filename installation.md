## Developer instalation
--------

### Download SDK man
-----

    curl -s "https://get.sdkman.io" | bash
    source "$HOME/.sdkman/bin/sdkman-init.sh"
    sdk version

Go to the directory:

    cd .sdkman/candidates

Install java

    Sdk install java

Install gradle

    Sdk install gradle

Install maven
    
    Sdk install maven

Install springboot

    Sdk install springboot
